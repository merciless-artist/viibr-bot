# VIIBR BOT
- Python 3.10+
- discord.py 2.6

Discord bot for the VIIBR/VIBE MUSIC Discord server (AI Umbrella)

Member features use slash commands, moderation and admin features use prefix $
- Two-tier help: `$help`/`$viibr` staff menu, `/help` member overview
- Ticket system (button panel, private ticket channels, transcript logging)
- Bulk message cleanup (`$delete`) and deletion logging (`$setlog`)
- Counting game (hard/easy modes, milestones with custom media)
- Birthday tracker (`/addmybd`, `/calendar`, daily announcements with rotating songs and a community wishes button)
- Resources (`/resources` links embed with admin management)
